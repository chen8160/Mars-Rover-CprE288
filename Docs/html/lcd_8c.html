void lprintf ( const char *  format,
  ... 
)

Prints a string to the lcd; Google "printf" for documentation.

Print a formatted string to the LCD screen. Mimics the C library function printf for writing to the LCD screen. The function is buffered; i.e. if you call lprintf twice with the same string, it will only update the LCD the first time.

Google "printf" for documentation on the formatter string.

Code from this site was also used: http://www.ozzu.com/cpp-tutorials/tutorial-writing-custom-printf-wrapper-function-t89166.html
